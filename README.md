# GLM Sniper 🎯

智谱 GLM5.1 Coding 包月套餐抢购助手 - Chrome 浏览器插件

## 功能

- 自动监控智谱套餐页面，检测 GLM5.1 Coding 包月套餐是否可购买
- 可配置检查间隔（默认 3 秒）
- 发现目标后桌面通知提醒
- 可选自动点击购买按钮
- 随机延迟防检测
- 实时日志查看
- 深色主题 UI

## 安装

1. 克隆本仓库
2. 打开 Chrome，访问 `chrome://extensions/`
3. 开启右上角「开发者模式」
4. 点击「加载已解压的扩展程序」
5. 选择本项目根目录

## 使用

1. 打开智谱的套餐购买页面（chatglm.cn 或 open.bigmodel.cn）
2. 点击浏览器工具栏中的插件图标
3. 配置参数（关键词、间隔、是否自动购买等）
4. 点击「开始抢购」
5. 插件会自动监控页面，发现可购买套餐时通知你

## 配置说明

| 选项 | 说明 | 默认值 |
|------|------|--------|
| 检查间隔 | 每次扫描的间隔秒数 | 3 秒 |
| 目标关键词 | 用于匹配目标套餐的关键词 | GLM-5.1, Coding, 包月 |
| 自动点击购买 | 发现目标后自动点击购买按钮 | 关闭 |
| 随机延迟 | 操作前加入随机延迟防检测 | 开启 |
| 桌面通知 | 发现目标时弹出系统通知 | 开启 |

## 注意事项

- 本插件仅供个人学习使用
- 自动购买功能仅点击购买按钮，实际支付仍需手动确认
- 如果页面结构变化，可能需要更新选择器配置
- 建议先在不开启自动购买的情况下测试

## 技术栈

- Chrome Extension Manifest V3
- Vanilla JavaScript
- Chrome Storage API / Alarms API / Notifications API
